<h1 align="center">Sail</h1>

<p align="center"><em>Sail Past Censorship</em></p>


Sail is a currently indev proxy by [RHW](https://github.com/rhenryw), [ThinLiquid](https://github.com/ThinLiquid), and [technonyte](https://github.com/technonyte00)

It will have a unique WebSocket system that will alowwed anybody to host the core of the site and publish a websocket that anybody can use on their Sail client.

We will have WebPorts of PC games, movies, shows, accounts, chat, and more!
